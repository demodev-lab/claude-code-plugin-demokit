# Glossary

용어 혼동을 줄이기 위한 demokit/bkit 용어집.

| 용어 | 의미 |
|---|---|
| PDCA | Plan-Do-Check(Analyze)-Act(Iterate) 루프 |
| Pipeline | 9단계 개발 파이프라인 상태머신 |
| Delegate Mode | 팀 실행을 단일 리더로 축소하는 모드 |
| Level Override | 레벨별 팀 정책 덮어쓰기 설정 |
| Level Profile Map | demokit 레벨 ↔ bkit 프로파일 호환 매핑 |
| Dynamic | bkit의 기본 실무 프로파일 (demokit에서 주로 Single/Multi/Monolith 대응) |
| Enterprise | bkit의 대규모 프로파일 (demokit에서 주로 MSA 대응) |
| Hook Runtime Toggle | 이벤트/스크립트별 hook 실행 on/off 제어 |
| Sticky Cache TTL | do 산출물 체크 결과를 일정 시간 재사용하는 TTL |
| Transition Hint | phase 전이 시 안내 메시지 |
