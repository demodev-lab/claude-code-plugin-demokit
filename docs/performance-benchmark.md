# Performance Benchmark Notes

## Suggested Baseline Metrics

- `task-completed.js` wall time
- `pipeline-ctl next` wall time
- PDCA do cycle latency (start -> next recommendation)

## Benchmark Tips

- warm/cold 구분
- 동일 feature 반복 측정
- hook enable 조합별 비교

## Report Template

- Environment
- Config snapshot
- Metric table
- Regression/Improvement summary
