# Core Mission

## 미션

> **Spring Boot 개발의 모든 반복 작업을 자동화하고, 일관된 품질을 보장하는 AI 개발 플러그인**

## 핵심 가치

### 1. Spring Boot 전문성
일반적인 코드 생성이 아닌, Spring Boot 생태계에 최적화된 패턴과 Best Practice를 적용한다.
- Spring Boot 3.5+ 최신 API 우선
- Deprecated API 자동 감지 및 대안 제시
- DRY 원칙 기반 코드 생성

### 2. PDCA 기반 품질 보장
Plan → Design → Do → Analyze → Iterate → Report 사이클을 통해 설계와 구현의 일치율(Match Rate)을 측정하고 자동 수정한다.
- Match Rate 90% 이상 목표
- 최대 5회 자동 반복 수정
- Gap 분석을 통한 누락/불일치 감지

### 3. 팀 에이전트 협업
15개 전문 에이전트가 역할 기반으로 협업하여 대규모 기능도 체계적으로 구현한다.
- 레벨별 팀 구성 (SingleModule → MultiModule → MSA)
- 오케스트레이션 패턴 (leader, council, swarm, pipeline, watchdog)
- 작업 의존성 기반 자동 분배

### 4. 컨텍스트 엔지니어링
프로젝트 분석, 패턴 학습, 메모리를 통해 일관된 코드를 생성한다.
- 세션 시작 시 프로젝트 자동 분석
- 기존 코드 스타일 감지 및 매칭
- 프로젝트 메모리를 통한 학습

## 대상 사용자
- Spring Boot 백엔드 개발자
- CRUD 중심 웹 애플리케이션 개발
- 개인 또는 소규모 팀 프로젝트
