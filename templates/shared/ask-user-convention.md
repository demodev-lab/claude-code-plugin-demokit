# AskUserQuestion 도구 사용 컨벤션

사용자에게 확인/선택 질문을 할 때는 **반드시 `AskUserQuestion` 도구를 사용**한다. 줄글 질문 금지.

## 규칙
1. 1회 호출에 최대 4개 질문을 묶어서 한 번에 질문한다.
2. `header`는 12자 이내 짧은 라벨 (탭에 표시됨).
3. 각 질문에 2~4개 선택지를 제공한다. 권장 옵션은 첫 번째에 배치하고 label 끝에 `(Recommended)`를 붙인다.
4. `description`에 각 선택지의 트레이드오프나 맥락을 설명한다.
5. 질문이 5개 이상이면 우선순위 높은 4개를 먼저 질문하고, 응답 후 나머지를 추가 호출한다.
6. 상호 배타적이지 않은 질문은 `multiSelect: true`를 사용한다.
7. 코드/레이아웃 비교가 필요한 질문은 `markdown` preview를 활용한다.
8. 단순 확인(y/n)도 AskUserQuestion으로 처리한다 — "진행" / "취소" 선택지 제공.
